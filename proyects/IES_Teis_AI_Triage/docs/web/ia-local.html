Uso de IA Local con Ollama

Para ejecutar el Agente de forma privada y sin costes, utilizaremos Ollama. Esta herramienta permite levantar modelos de lenguaje grandes (LLMs) directamente en nuestro equipo.

1. Instalar Ollama

Descarga el instalador según tu sistema operativo desde la página oficial (ollama.com) o instálalo por terminal en Linux/macOS:

# Instalar en Linux/macOS
curl -fsSL https://ollama.com/install.sh | sh

2. Descargar y Ejecutar

Abre una nueva terminal y ejecuta el siguiente comando para descargar e iniciar Llama 3 (un modelo muy capaz creado por Meta):

# Descarga y ejecuta Llama 3 (8B)
$ ollama run llama3

*La primera vez tardará unos minutos en descargar el modelo (~4.7GB).
